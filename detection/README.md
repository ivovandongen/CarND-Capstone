# Tools and data to train a detection model


## Requirements

- python 2.7
- protoc3
- a nvidia gpu that works with tensorflow

## Installation

- Optionally: ./install-protobuf3.sh
- ./install.sh
- source setup.bash
- ./test-env.sh

## Usage

- source setup.bash
- ./prepare_data.sh
- ./train_simulator.sh
